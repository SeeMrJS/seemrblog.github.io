### javascript 中的原型
JavaScript中的最开始了继承理念是原型继承，如下:
```js
function aClass ({age,name,gender}) {
   this.age = age
   this.name = name
   this.gender = gender
}
const obj = new aClass({
  name:'seemr',
  age:22,
  gender:2
})
aClass.prototype.info = 'hello woeld'
aClass.prototype.say = () => 'hello world!'

obj.info
obj.say()
```
